# Следы арабской культуры в названиях современных блюд России

Презентация исследовательской работы по этимологии 12 ключевых кулинарных терминов русского языка.

**Автор:** Исмаилов Абдула Таусович, 10 класс
**Школа:** ГБОУ Школа №1550, Москва
**Год:** 2026

## Открыть презентацию

После публикации репозитория на GitHub Pages презентация будет доступна по адресу:

```
https://<твой-логин>.github.io/<название-репозитория>/
```

## Управление

- `←` / `→` — переключение слайдов
- `Пробел` — следующий слайд
- `Home` / `End` — первый / последний слайд
- Клик по точкам внизу — переход к слайду
- Кнопки `◀ ▶` — навигация мышью

## Структура

- `index.html` — единый файл презентации (HTML + CSS + JS)
- 13 слайдов с двумя интерактивными диаграммами (Chart.js)
- Шрифты Google Fonts (Cormorant Garamond + Manrope)
- Изображения с Unsplash

## Технологии

- HTML5 / CSS3 (без фреймворков)
- [Chart.js 4.4](https://www.chartjs.org/) — диаграммы
- [Google Fonts](https://fonts.google.com/) — типографика
- [Unsplash](https://unsplash.com/) — фоновые изображения

Для работы нужен интернет (подгружаются шрифты, диаграммы и фото).
